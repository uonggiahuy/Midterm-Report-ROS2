# Midterm-Report-ROS2
Báo cáo giữa kỳ môn Lập trình Robot với ROS tại UET-VNU

Cấu trúc package của Littlebot
littlebot/

├── CMakeLists.txt

├── package.xml

├── meshes/        (Chứa các file .stl của littlebot)

├── little_bot.urdf 

├── launch/        (Chứa file gọi cả hệ thống lên)

├── rviz/          (Chứa file config giao diện RViz)

└── scripts/       (Chứa code Python điều khiển tay máy)
